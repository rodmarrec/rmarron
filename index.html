the Hoptimist

Full Stack Web Application

JavaScript | MongoDB | Express | Node.js | HTML | CSS + Bulma | Heroku

Paired up with a fellow developer to create an app for creating craft beer reviews on-the-go. With the Hoptimist you can add a new Brewery or select one that exists in the database, and then either choose from the list of craft beers or add a new one in order to review it! My partner and I completed this project in 5 days with him focusing on the backend, and I on the front.

Grapevine

Full Stack Web Application

JavaScript | MongoDB | Express | Node.js | HTML | CSS + Bootstrap

Solo project involving development of a full stack web application from inception with wireframes and ERD models to completion of an MVP.

The concept behind Grapevine was born from identifying gaps within LinkedIn's platform where often the job seeker is jumping into an ocean of job posts and must depend on their own experience and judgement to weed through hundreds of posts. Grapevine enables users to share an existing job post from a different website and provide TODOs for other users; whether it's a red flag, a clarification, or providing resources. This app aims to create community and solidarity between job seekers so as to help and build a network.

Wayfarer

Full Stack Web Application

Python | PostgreSQL | Django | HTML | CSS + Bootstrap |Heroku

Project team of four devs was tasked with working with a client to create a travel-based blog web app. Decisions made throughout the development of the MVP had to be figured out by both my dev team alongside the client.

I contributed to leading the team by establishing clear communication and collaboration workflows, along with design and implementation of the frontend. The client provided conceptual look and feel with their color palette of choice in the brief.

Detailed pre-programming planning set us up for success. We were able to achieve MVP with stretch goals, and deploy our app ahead of schedule.

Catch'Ms

Speed Memory Game

JavaScript | HTML | CSS

Project involved developing a game by using only Javascript in five days. I chose to do Memory with the added challenge of a tight time constraint and having to match icons that strongly resemble each other.

This game is more challenging than it looks! I welcome you to try it out.
I've managed to beat it only a handful of times-- and I created it!

Best of Luck and go match-and-catch those pesky Ms!
